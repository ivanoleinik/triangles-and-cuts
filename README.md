# triangles-and-cuts
Geometry task solution

## Examples of usage

### Solve problem for equilateral triangle
```
make
./triangles -eq
x1 y1 x2 y2 x3 y3 l
```

### Solve problem for acute triangle
```
./triangles
x1 y1 x2 y2 x3 y3 l
```

### Run tests
```
make test_triangles
./test_triangles
```
If nothing happened, all tests passed.

## About solution
Time complexity is O(1).
